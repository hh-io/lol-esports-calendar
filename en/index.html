League of Legends Esports Schedule Calendar (.ics)

LPL · LCK · LEC · LCP · LTA · MSI · Worlds — subscribe once, refreshed hourly, works with Apple Calendar / Google Calendar / Outlook.

This is a free, open-source League of Legends esports schedule calendar subscription service. It automatically fetches the latest schedules for LPL (China), LCK (Korea), LEC (Europe), LCP (Pacific), LTA (Americas), and international events like MSI, Worlds, First Stand and EWC, generating standard .ics calendar files hosted on GitHub Pages and refreshed hourly by GitHub Actions. Scores appear on finished matches within roughly 1–2 hours after they end.

🍎 How to subscribe

Apple Calendar (macOS): Calendar app → menu File → New Calendar Subscription… → paste the URL above → set "Auto-refresh" to "Every hour".

iPhone / iPad: Settings → Calendar → Accounts → Add Account → Other → Add Subscribed Calendar → paste the URL. Replacing https:// with webcal:// lets you open the calendar app directly from the link.

Google Calendar (web): "Other calendars" → "+" → "From URL" → paste the URL.

Outlook: "Add calendar" → "Subscribe from web" → paste the URL.

❓ FAQ

Are match times in my local time zone? Match times are stored in UTC; your calendar client converts them to your local time zone automatically.

Why is a region's calendar empty? A region with no upcoming matches during the off-season yields an empty calendar — this is normal and events reappear once new schedules are published.

Where does the data come from? From the public endpoint the lolesports website itself uses. This project only reformats the data and has no affiliation with or endorsement from Riot Games.
