무료 오픈소스 리그 오브 레전드 e스포츠 일정 캘린더 구독 서비스입니다. LPL(중국), LCK(한국), LEC(유럽), LCP(태평양), LTA(아메리카)와 MSI, Worlds, First Stand 등 국제 대회의 최신 일정을 자동으로 수집해 표준 .ics 캘린더 파일을 생성하고, GitHub Pages에서 호스팅하며 GitHub Actions가 매시간 갱신합니다. 경기 종료 후 약 1~2시간 내에 해당 경기에 스코어가 표시됩니다.

🍎 구독 방법

Apple 캘린더 (macOS): 캘린더 앱 → 메뉴 파일 → 새로운 캘린더 구독… → 위 URL 붙여넣기 → "자동 새로 고침"을 "매시간"으로 설정.

iPhone / iPad: 설정 → 캘린더 → 계정 → 계정 추가 → 기타 → 구독 캘린더 추가 → URL 붙여넣기. https://를 webcal://로 바꾸면 링크를 눌러 바로 캘린더 앱을 열 수 있습니다.

Google 캘린더 (웹): "다른 캘린더" → "+" → "URL로 추가" → URL 붙여넣기.

Outlook: "캘린더 추가" → "웹에서 구독" → URL 붙여넣기.

❓ 자주 묻는 질문

시간은 내 현지 시간대로 표시되나요? 경기 시간은 UTC로 저장되며 클라이언트가 자동으로 현지 시간대(한국 표준시 등)로 표시합니다.

비시즌에는 캘린더가 비어 있나요? 특정 지역에 예정된 경기가 없으면 빈 캘린더가 되는 것이 정상이며, 새 일정이 공개되면 자동으로 나타납니다.

데이터 출처는 어디인가요? lolesports 공식 웹사이트가 자체적으로 사용하는 공개 API에서 가져오며, 본 프로젝트는 형식 변환만 수행하고 Riot Games와 제휴하거나 보증 관계에 있지 않습니다.
