英雄联盟职业赛程日历订阅（.ics）

LPL · LCK · LEC · LCP · LTA · MSI · Worlds — 一键订阅，每小时自动更新，兼容 Apple 日历 / Google 日历 / Outlook。

这是一个免费开源的英雄联盟职业联赛赛程日历订阅服务。它自动抓取 LPL（中国）、LCK（韩国）、LEC（欧洲）、LCP（太平洋）、LTA（美洲）以及 MSI、Worlds（S 赛）、First Stand 等国际赛的最新赛程， 生成标准 .ics 日历文件，托管在 GitHub Pages 上，由 GitHub Actions 每小时刷新一次。比赛打完后约 1～2 小时内，对应场次还会带上比分。

🍎 如何订阅

Apple 日历（macOS）：日历 App → 菜单栏「文件」→「新建日历订阅…」→ 粘贴上面的 URL → 把「自动刷新」设为「每小时」。

iPhone / iPad：设置 → 日历 → 账户 → 添加账户 → 其他 → 添加已订阅的日历 → 粘贴 URL。把 https:// 换成 webcal:// 可直接点链接唤起日历 App。

Google 日历（网页版）：左侧「其他日历」→「＋」→「通过网址添加」→ 粘贴 URL。

Outlook：「添加日历」→「从 Internet 订阅」→ 粘贴 URL。

🌍 多语言 / Multi-language / 다국어

日历正文支持三种语言：简体中文（/dist/）、 English（/dist/en/）、 한국어（/dist/ko/）。例如英文版 LPL 日历为 https://hh-io.github.io/lol-esports-calendar/dist/en/lpl.ics。

❓ 常见问题

时间是本地时区吗？ 比赛时间以 UTC 存储，客户端会自动按你的本地时区（如北京时间）显示。

休赛期日历是空的？ 某赛区暂无未来赛程时为空日历，属正常现象，新赛程发布后会自动出现。

数据来源？ 来自 lolesports 官网自身使用的公开接口，本项目仅做格式转换，与 Riot Games 无隶属或背书关系。
